# 所用技术栈

## 后端

- Spring
- SpringMVC
- MyBatis
- SpringCloud-Alibaba-All(Sentinel，GateWay, Nacos，Feign)
- SpringFramework
- OAuth2
- Jwt
- SpringSecurity
- MyBatisPlus
- Maven >= 3.0
- Java11
- Redis

## 前端

- Nuxt
- Vue
- Element-UI
- Vue-Element-Admin
- Vue-Cli

## 数据库

- MySql8.0

## 代理

- Nginx
- 内网穿透

## 工具类

- HTTPClient
